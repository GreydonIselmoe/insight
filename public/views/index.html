About

Credits [CRE] is the first successful "soft" side-chain implementation coin, built off of the Bitcoin Blockchain.

It features a never before seen Proof of Work / Proof of Deposit mining system which allows miners to prepare mining deposits in order to achieve more efficient mining. This feature insures that not only raw hashing power alone can be used to a miners advantage, and after block 33000 full migration to the Proof of Deposit system will occur and the network will require Credits [CRE] mining deposits in order to achieve maximum mining efficiency.

Credits [CRE] is an open sourced coin and all source code is available via github.com: Credits [CRE] source code

This block explorer is still in development, we are working on making the recent transactions functional, as well as providing address-lookup capabilities.
